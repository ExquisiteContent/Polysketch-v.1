Welcome to Polysketch! A web program used to create artwork using precise polygons and shapes generated on the HTML Canvas element.

Use the tools under the canvas to get started. You can customize nearly everything! To generate a shape, you only need to specify the sides in the textbox. Also try switching up the angle, size, or origin. Take note of the scale along the grid axes.

You can also change the color and width of the shapes lines.

As of yet there is no function to fill in space, rather this can be done with dense lines drawn where needed.

Text inputs DO NOT take arithmetic operations as valid inputs i.e.(13+29).
